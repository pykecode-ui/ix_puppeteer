Como Adicionar um Novo Bot
Instruções rápidas
  1. Copie a pasta bot/ para o PC ou VPS onde o ixBrowser está instalado.
  2. Edite bot/config.js e defina DASHBOARD_API_URL com o endereço deste servidor.
  3. Execute npm install uma única vez na pasta bot/.
  4. Inicie o bot com node bot.js.
  5. O bot aparecerá automaticamente neste painel!
