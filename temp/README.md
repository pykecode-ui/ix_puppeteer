# Pasta Temporária e de Testes

**REGRA DO PROJETO:**
Todo e qualquer arquivo criado para fins de testes, scripts temporários, rascunhos ou integrações não oficiais (como o `adspower.py` e scripts de extração) deve ser colocado **exclusivamente** nesta pasta `temp/`.

Essa regra garante que a raiz do projeto e os módulos principais fiquem limpos, organizados e sem código morto.
